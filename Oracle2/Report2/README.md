### awr 주기 설정
```sql
--RETENTION : 보관시간, INTERVAL : 주기
--설정단위 : 분
EXEC DBMS_WORKLOAD_REPOSITORY.MODIFY_SNAPSHOT_SETTINGS(RETENTION=>60*24*14,INTERVAL=>15);
```
