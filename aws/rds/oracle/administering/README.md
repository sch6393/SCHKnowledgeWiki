### RDS에서 마스터 사용자에게 부여되지 않는 권한
|PRIVILEGE|
|-|
|ALTER DATABASE|
|ALTER DATABASE LINK|
|ALTER PUBLIC DATABASE LINK|
|ALTER SYSTEM|
|CREATE ANY DIRECTORY|
|DROP ANY DIRECTORY|
|EXEMPT ACCESS POLICY|
|EXEMPT IDENTITY POLICY|
|GRANT ANY PRIVILEGE|
|GRANT ANY ROLE|
|SYSDBA|
|SYSOPER|
>해당 권한을 사용하고 싶다면 위의 메뉴얼을 참조해 프로시저 형태로 실행시켜야 함
